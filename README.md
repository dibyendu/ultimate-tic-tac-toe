### To train the agent

- set the training parameters in [Train.mjs](#AlphaZeroAgent/Train.mjs)
- `node Train.mjs`

### To run the app

- `npm run build`
- `npm run serve`

### Development

Run `npm start`, edit [client/Landing.jsx](#client/Landing.jsx) and watch it live update!
